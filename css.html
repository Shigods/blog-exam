What is CSS?

CSS (Cascading Style Sheets) is the language used to style and format HTML content. It controls colors, fonts, layout, and responsiveness.

CSS Example


h1 {
  color: blue;
  text-align: center;
}

p {
  font-size: 16px;
  line-height: 1.5;
}

Ways to Use CSS

  • Inline CSS: Directly inside an HTML tag using the style attribute.
  • Internal CSS: Inside a <style> tag in the HTML document.
  • External CSS: In a separate .css file linked to your HTML.

Why Learn CSS?

CSS brings your pages to life. With it, you can design layouts, animations, and responsive websites that look great on all devices.

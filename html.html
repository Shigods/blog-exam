What is HTML?

HTML (HyperText Markup Language) is the standard foundation of every website pages. It uses tags to describe the structure and content of a website. With HTML, you define headings, paragraphs, links, images, and more.

Basic Structure


<!DOCTYPE html>
<html>
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Welcome to my site</h1>
    <p>This is a simple HTML example.</p>
  </body>
</html>

Common HTML Tags

  • <h1> to <h6> — Headings for titles and subtitles
  • <p> — Paragraph text
  • <a> — Hyperlinks to other pages
  • <img> — Embed images
  • <ul> / <ol> — Lists of items
  • <table> — Create tables
  • <form> — Collect user input

Why Learn HTML?

HTML is the first step in web development. Once you master it, you’ll be ready to move on to CSS for design and JavaScript for interactivity.

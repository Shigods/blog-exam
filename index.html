Web Development Guide

Learn the foundations of HTML, CSS, Git, and Tailwind CSS

HTML: The Structure of the Web

HTML, or HyperText Markup Language, defines the basic structure of every website. It uses tags to represent elements like text, images, and links. Each HTML page begins with a doctype declaration and contains a head and body section that organize its content.


  • Headings: Represented by <h1> to <h6>.
  • Paragraphs: Written inside <p> tags.
  • Links: Created using the <a> tag.
  • Images: Displayed using the <img> tag.

HTML provides the foundation that every web developer builds upon. Learn more from MDN Web Docs.

CSS: Styling the Web

CSS, or Cascading Style Sheets, controls the appearance of a website. It defines colors, fonts, spacing, and layout. While HTML focuses on structure, CSS focuses on presentation and design. It can be written inline, internally within a <style> tag, or externally in separate files.

  1. Selectors identify which elements to style.
  2. Properties define what to change (e.g., color, margin).
  3. Values specify the style (e.g., blue, 20px).
Example:
h1 { color: lightgreen; text-align: center; }

With CSS, developers can transform plain HTML into visually stunning, user-friendly websites.

Git: Tracking and Managing Code

Git is a version control system that helps developers manage and track changes to code. It allows collaboration, branching, and versioning, ensuring that project history is preserved and reversible when necessary.

Command Description
git init Initialize a new Git repository
git add . Stage all changes for commit
git commit -m "message" Save a snapshot of changes
git push Upload commits to a remote repository

Git enables efficient teamwork by allowing multiple developers to work on the same project without conflict.

Tailwind CSS: Utility-First Design

Tailwind CSS is a utility-first CSS framework that provides prebuilt classes for rapid UI design. Instead of writing traditional CSS rules, you apply ready-to-use utility classes directly in your HTML code.

Why Developers Use Tailwind CSS

  • Speeds up styling by using class-based utilities.
  • Encourages consistent and responsive design patterns.
  • Integrates seamlessly with frameworks like React and Vue.

Example: class="bg-blue-500 text-white p-4 rounded"

Learn more at tailwindcss.com.

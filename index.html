HTML Basics

HTML stands for HyperText Markup Language. It defines the structure of web pages.
Below are the main topics:

  • Document structure
  • Text and content tags
  • Lists and layouts
  • Tables and forms

CSS Styling

CSS enhances the visual design of web pages.

  1. Selectors define elements to style.
  2. Properties modify appearance.
  3. Values determine how those properties behave.
<div> containers and <span> elements help structure and style content.

Git Essentials

Git is a version control system used by developers to manage code changes.

Command Description
git init Initialize repository
git add Add files to staging
git commit Save changes
